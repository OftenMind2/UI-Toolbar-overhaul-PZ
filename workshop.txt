description=A modern, modular, and universally compatible UI framework that replaces the standard Project Zomboid tooltips with a sleek Tactical PDA aesthetic.

[b]Features:[/b]
* [b]Tactical PDA Aesthetic:[/b] Smooth slide-in animations, dynamic measurements, and an immersive dark-obsidian background.
* [b]Crash-Resistant:[/b] Item data is read via reflection (capability checks), so modded items that are missing expected methods degrade gracefully instead of throwing Java exceptions.
* [b]Universal Context Wrapping:[/b] Applies seamlessly to Inventory, Crafting, Vehicles, and World Objects.
* [b]Compatibility:[/b] Coexists with Inventory Tetris, Equipment UI, Brita's Weapon Pack, KI5 Vehicles, and TooltipLib without hardcoded patches.

Made for Build 42.
